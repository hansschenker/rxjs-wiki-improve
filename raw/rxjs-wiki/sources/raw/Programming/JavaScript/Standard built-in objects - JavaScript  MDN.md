Detailed summary

- The [[JavaScript]] language has a set of standard built-in objects that are categorized into various groups, including value properties, function properties, fundamental objects, error objects, and more, which provide a wide range of functionalities and features.
- The value properties in JavaScript include globalThis, Infinity, NaN, and undefined, which return a simple value and have no properties or methods, making them basic yet essential components of the language.
- JavaScript also has function properties, such as eval(), isFinite(), isNaN(), parseFloat(), parseInt(), decodeURI(), decodeURIComponent(), encodeURI(), encodeURIComponent(), escape(), and unescape(), which are global functions that directly return their results to the caller and are used for various purposes like parsing, encoding, and decoding.
- The fundamental objects in JavaScript represent basic language constructs, including Object, Function, Boolean, and Symbol, which are the building blocks of the language and are used to create and manipulate other objects.
- Error objects are a special type of fundamental object in JavaScript, and they include the basic Error type, as well as several specialized error types, such as AggregateError, EvalError, RangeError, ReferenceError, SuppressedError, SyntaxError, TypeError, URIError, and InternalError, which are used to handle and manage errors in the language.
- JavaScript also has objects for numbers and dates, including Number, BigInt, Math, Date, and Temporal, which provide a range of mathematical and date-related functionalities and are used for tasks like calculations, conversions, and formatting.
- The text processing objects in [[JavaScript]], such as String and RegExp, represent strings and support manipulating them, making it easy to work with and process text data in the language.
- Indexed collections in JavaScript, including Array, TypedArray, and various types of arrays like Int8Array, Uint8Array, and Float64Array, represent collections of data that are ordered by an index value and are used for storing and manipulating data in a structured way.
- Keyed collections, such as Map, Set, WeakMap, and WeakSet, represent collections that use keys and contain elements that are easily iterated in the order of insertion, making them useful for tasks like data storage, retrieval, and manipulation.
- JavaScript also has objects for structured data, including ArrayBuffer, SharedArrayBuffer, DataView, Atomics, and JSON, which represent and interact with structured data buffers and data coded using [[JSON | JavaScript Object Notation]] (JSON), enabling efficient data storage and exchange.
- The language has objects for managing memory, such as WeakRef and FinalizationRegistry, which interact with the garbage collection mechanism and help manage memory usage and optimization.
- Control abstraction objects in JavaScript, including Iterator, AsyncIterator, Promise, GeneratorFunction, AsyncGeneratorFunction, Generator, AsyncGenerator, AsyncFunction, DisposableStack, and AsyncDisposableStack, help structure code, especially async code, and make it easier to write and manage complex asynchronous operations.
- The Reflect and Proxy objects in [[JavaScript]] provide reflection capabilities, allowing developers to inspect and manipulate objects, and control the behavior of other objects, making it easier to write flexible and dynamic code.
- Finally, JavaScript has internationalization objects, such as Intl, Intl.Collator, Intl.DateTimeFormat, Intl.DisplayNames, Intl.DurationFormat, Intl.ListFormat, Intl.Locale, Intl.NumberFormat, Intl.PluralRules, Intl.RelativeTimeFormat, and Intl.Segmenter, which provide additions to the [[ECMAScript]] core for language-sensitive functionalities and enable developers to create localized and culturally sensitive applications.
