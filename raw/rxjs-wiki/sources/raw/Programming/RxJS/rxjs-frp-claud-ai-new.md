## Introduction to Functional Reactive Programming
- Functional programming principles include immutability, pure functions, function composition, and higher-order functions
- Reactive programming concepts involve asynchronous data streams, the [[Observer pattern | observer pattern]], and event-driven systems
- Functional Reactive Programming (FRP) combines functional programming with reactive programming, providing benefits like composability, declarative code, and unified error handling

## RxJS Overview and Features
- RxJS is a library for reactive programming using Observables, allowing for the composition of asynchronous code and providing powerful operators for transforming streams
- RxJS operators work across different domains, creating a universal vocabulary for reactive programming
- RxJS provides a unified API for handling async operations, a rich operator library, and built-in error handling and memory management mechanisms.
- The library has a strong theoretical foundation in functional programming, rooted in concepts like monads, functors, and category theory.

## RxJS Components and Testing
- Key components of RxJS include Observables, Observers, Operators, Subscriptions, Subjects, and Schedulers, which work together to enable reactive programming in JavaScript.
