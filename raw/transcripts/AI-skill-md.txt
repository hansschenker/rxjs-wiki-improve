# Build Web UI Component

Expert skill for building UI components in the OpenLifeLog Next.js web application following established patterns and conventions.

## Tech Stack

- ** Framework **: Next.js 15 (App Router), React 19
- ** UI Library **: Shadcn/ui components
- ** Styling **: Tailwind CSS 4
- ** API Client **: `@openlifelog/sdk'
- ** State **: React hooks + Context API
- ** Auth **: AuthContext with JWT tokens
