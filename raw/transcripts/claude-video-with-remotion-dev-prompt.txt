API Configuration

. API Key: Set via environment variable KIE_API_KEY

. Base URL: https://api.kie.ai/api/v1/jobs

Security Note: Never hardcode API keys in project files. Store your key in a .env file
or export it as an environment variable:

export KIE_API_KEY-'your-api-key-here'
